## 一、分类

```java
// 属性：表示人和物具有的属性，可以加very
beautiful: very beautiful
// 类别：表示事物的类别, 不可以加very
Chinese、empty
// 颜色： 浅色和深色
浅色：light(光)、pale(微弱，苍白)、bright(明亮)
深色：dark(黑暗)、deep（深）
// 强调
absolute(绝对)、utter（彻底）、total（全部）、entire（整个）
// -ing和-ed结尾：某人某事某种情况的特点表示令人感觉
boring（无聊）
// 复合：形容词+名词+ed，形容词/副词+过去分词、形容词/副词/名词+现在分词、名词+形容词
```

## 四、比较级

### 4.1 规则变化

```java
// 单音节词和少数双音节词。比较级加er 最高级加est
warm(温暖)  warmer（更温暖） warmest（最温暖）
// 元音加上辅音字母结尾，双写辅音字母。比较级加er，最高级加est
big(大)  bigger（更大） biggest（最大）
// e结尾。比较级加r，最高级加st
large larger largest
// y结尾，变y为i。比较级加er，最高级加est
happy happier happiest
// 多音节词 单数前面比较级加more，最高级加most
fantastic > more fantastic > most fantastic
```

### 4.3 用法

```java
// 原级：没有比较的意思 或 原级比较表示和什么一样（as...as / the same as / the identical with）
That is good(那很好)
// 比较级：能单独使用，前面能加修饰词（a lot, much, rather, a little, even等），与than连用比较两者
You should be more patient(你应该跟耐心)
I swim faster than you(我游的比你快)
// 最高级：前面要加the，能单独使用，前面能加修饰词（far,mostly, very, much等）,at + 最高级表示处于最什么状态
That is the best chiose（那个最好的选择）
The car power will at it best(这车的动力将达到最佳状态)
```

### 4.4 特殊结构

```java
// 比较级and比较级：表示越来越
faster and faster(越来越快)
// 多音节单词：单词可以只写一个，放在最后面
more and more fantastic(越来越了不起)
// the比较级...the比较级： 越怎么越怎么
The richer, the more badass(越有钱越无法无天)
// more than + 名词 = not only(不仅，不知)
we are more than friend(我们不只是朋友)
// more than + 数词 = very（非常）
We are more than interested in search gold(我们对搜索黄金非常感兴趣)
// more ... than: 第一重意思是比什么多，第二重意思是与其什么不如
My video is more interesting than your video(我的视频比你的视频有趣多了)
You are more give up than painful(你与其痛苦不如放弃)
// no more than: 只有，仅仅（等于的意思）
I have no more than four games(我只有四款游戏)
// less than: 少于
We leave in less than a week(我们不到一周就走)
// no less than: 多达
The total amount no less than 5000 dollars(总金额多达5千美元)
// not less than: 不少于
I have not less than four games(我有不少于四款游戏)
// no less ... than： 和什么一样
I am no less careful than you(我和你一样小心)
```

### 4.5 易混

```java
// too much（表太多，修饰名词表数量）, much too（表过分，修饰形容词或副词）
I have too much money(我有太多钱了)
That is much too expensive(这贵的过分了)

// good（用在定语或者表语）、well（用在状语）
It is good 表语(他很好)
Study well and improve every day状语(好好学习天天向上)

// nice（愉快，指外表）, fine（身体好，天气好）
nice kill(好击杀)
I am fine(我没事)

// lonely(孤独的，真的孤独), alone（独自，不一定孤独）
The lonely student is starving for friendship(那个孤独的学生渴望友情)
I live alone(我一个人住)

// quick（速度快），fast（速度快），soon（时间快）
He is a quick man(他是一个快男)
come soon(很快到来)

// interesting（令...感兴趣）, interested(对...感兴趣)
The book is very interesting(这些书很有趣)
They are interested in English(他们对英语感兴趣)

// living（活着的，存在的，一摸一样的，强烈的）, alive（活着的人）, live（活着的东西）, lively（活泼的、生动的）
The living polymer is high technology(这高活性聚合物是高科技)
John is alive yet(约翰还活着)
A live album(一个现场专辑)
Her lively mind(他灵活的头脑)
```
